User Guide

What this tool does

From two inputs — a state and a county — it generates a 4-page mailing package you can print and mail to a U.S. county sheriff: a personalized cover letter, a one-page county savings report, and a 2-page constitutional brochure.

How to use it

1. Enter your name, email, mailing address, and phone. Click Save My Info so you never re-type it.
2. Click Continue. Pick a state, then a county. The sheriff's office address auto-fills from the embedded database.
3. (Optional) If you've looked up the sheriff's name — e.g., from the Google search link beside the field — paste it into the Sheriff Name box. If you leave it blank, the letter is addressed to the office and opens "Dear Sheriff,".
4. Click Preview & Print. The 4 pages display on screen so you can review them. Use Ctrl+P (Windows) or Cmd+P (Mac) — or the toolbar button — to print or save as PDF.

Window-envelope layout

The cover letter is laid out for #10 window envelopes: your return address is right-justified at the top, the date is centered, and the target sheriff address occupies the left 60% of its row (with the "Re:" subject line on the right 40%). When the letter is folded into thirds and inserted into a #10 window envelope, only the target sheriff address shows through the window.

Verifying or correcting an address

Below each address you'll see two links: "Verify this address on Google" opens a Google search so you can confirm the address against official sources; "Report wrong address" opens a pre-filled email to roleigh@pobox.com so the master GitHub package can be corrected for everyone.

Address confidence dots

Green — verified from an authoritative source.
Yellow — from a single source; spot-check recommended.
Blue — legacy entry; verify before mailing.
Red — flagged as possibly outdated; verify before mailing.
Gray — address not yet in the database; please look it up before mailing.

Excluded states

Five states are excluded because they do not have an elected, constitutionally sovereign county sheriff with the standing established in Printz v. United States (1997): Connecticut (sheriff abolished 2000), Alaska (no sheriff office), Hawaii (sheriffs are appointed state employees), Rhode Island (sheriffs appointed by governor), and Delaware (no county sheriff offices).

How the savings report works

Every figure is calculated from one input — the county's estimated population (U.S. Census Vintage 2024). All figures are modeled estimates scaled from a reference county (Boone County, KY, pop. 136,000), validated against Boone's FY2024 budget data. No actual fleet data is obtained from any county; this is disclosed on every page.

Editing the cover letter

The cover letter is editable in the print preview — click anywhere in the letter text to make changes before printing. The savings report and brochure pages are not editable.
