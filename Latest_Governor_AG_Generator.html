Background: Why This Tool Was Created

This tool is a companion to the Sheriff Mailing Package Generator. Reader feedback requested brochures aimed at governors and state attorneys general, who have even stronger standing than sheriffs to bring a constitutional challenge against post-1970 federal regulations that block remanufacturing of patent-expired designs.

The U.S. Constitution’s Patent Clause (Article I, Section 8, Clause 8) grants patent protection for “limited Times” — meaning that after a patent expires, the design enters the public domain for the public to use freely. But energy and safety regulations enacted after 1970 make it categorically impossible to remanufacture any vehicle or machine whose patents have expired. The designs never enter a functional public domain. This is costing American households an estimated $2,420 per year.

Governors can direct their attorneys general to evaluate and file challenges, issue executive orders, and build multi-state coalitions. Attorneys general have three independent standing theories (parens patriae, proprietary/fiscal, sovereign) and access to the proven multi-state AG action model used in Texas v. EPA and ACA challenges.

Key concept: competitive pressure on new items. A state saves even when buying only new equipment. When remanufactured alternatives exist, manufacturers must lower new-item prices to compete — exactly as brand-name drug prices fall 30–80% when generics enter the market.

What This Tool Generates

From a state selection and office type choice, the tool generates a 4-page mailing package:

  1. Cover Letter (1 page) — personalized with your contact info, the official’s name and office address, and state-specific Floor and Ceiling savings figures. The Re: line uses the Floor (“Minimally $X Million”); the grey box shows both.
  2. State Procurement Savings Report (1 page) — Census Bureau data showing construction spending, equipment costs, and a dual Floor/Ceiling savings table. The Floor (15%, green) is savings from competitive pressure even buying new. The Ceiling (45%, navy) is savings from purchasing remanufactured directly.
  3. Constitutional Brochure (2 pages) — the full Patent Clause analysis, international evidence, standing theories, and call to action — customized for governors or attorneys general.

How to Use This Tool (v4.0)

  1. Open the HTML file by double-clicking it in any browser. No account or internet connection required for single-state use.
  2. Choose Governor or Attorney General as the target office.
  3. Enter your personal information and click “Save My Info for Next Time.”
  4. Click the generate button to proceed to state selection.
  5. Select a state — the official’s name and address auto-fill instantly from embedded data (updated March 2026). Fields remain editable.
  6. Verify if needed — click the “Verify with Google Search” link to confirm the name and address are current.

Important: When editing the official’s name, enter just the name (e.g., “Andy Beshear” not “Governor Andy Beshear”). The cover letter automatically adds the title.

Batch Mode

Generate packages for multiple states in one session:

  1. Select a state — name and address auto-fill.
  2. Click “Add to Batch” to queue it.
  3. Repeat for as many states as desired.
  4. Click “Preview & Print Entire Batch” — all states render in sequence, 4 pages each.

Note: All entries in a single batch must be the same office type (all governors or all AGs). To switch office types, return to the first page.

Editing the Cover Letter

The cover letter is fully editable in the preview. Click anywhere on the letter text to make changes before printing. The savings report and brochure pages are not editable in preview, but downloaded HTML files can be opened in a browser and the cover letter edited there.

Saving Your Work

Click “Save & Print Pages” to download one .html file per state: ST-Governor-Package.html (or ST-AG-Package.html). Each file contains all 4 pages. For a batch of 5 states, 5 files download.

Printing: Open the HTML File in Your Browser

Double-click the downloaded .html file to open it in your browser, or drag it onto a browser window. Then press Ctrl+P. In the print dialog, set Margins to “None,” enable “Background graphics,” and use US Letter paper size. Do not open these files in Microsoft Word — Word does not support CSS grid and will collapse the brochure columns.

Office Type Differences

Governor brochure emphasizes what a governor can do: direct the AG, issue executive orders, build coalitions, use the bully pulpit. The cover letter focuses on procurement savings and cost of living. Both the Staggering Scope and Remedy sections reinforce that states save even buying new, because competitive pressure lowers all prices.

AG brochure emphasizes legal theory: belt-and-suspenders litigation strategy, favorable case law, filing roadmap, and why this has never been raised. The “Why Never Raised” section addresses the “who would buy that?” misconception — remanufactured alternatives lower new-item prices for everyone. The cover letter frames this as a question of first impression.

Both brochures share the same international evidence, standing theories, remedy sections, and About the Author box.

✅ ZIP Downloaded Successfully

💻 Windows (if you see “These files can’t be opened”):
  1. Open File Explorer and navigate to your Downloads folder (or wherever the ZIP was saved).
  2. Right-click the ZIP file (e.g., All50-Governor-Packages.zip).
  3. Click Properties.
  4. At the bottom of the General tab, check the box next to “Unblock”.
  5. Click Apply, then OK.
  6. Now double-click the ZIP to open it with File Explorer or WinZip.
🍎 Mac (if you see “cannot be opened because it is from an unidentified developer”):
  1. Control-click (or right-click) the ZIP file and choose Open.
  2. If that doesn’t work: go to System Settings → Privacy & Security, scroll down, and click “Open Anyway” next to the blocked file.

This is a standard security measure for files downloaded from the internet. It does not indicate any problem with the file.
